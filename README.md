# Retail Sales Insights Dashboard

This project demonstrates practical data analysis skills through a comprehensive examination of retail sales data. The analysis focuses on exploratory data analysis, key performance indicators, and actionable business insights.

## Overview

The project showcases essential data analyst capabilities including data cleaning, statistical analysis, and business intelligence reporting. Using Python and SQL, it transforms raw sales data into meaningful insights that drive business decisions.

**Core Components:**
- Comprehensive exploratory data analysis
- Three critical business KPIs: Total Revenue, Average Order Value, and Category Performance
- Monthly revenue trend analysis
- Interactive data visualizations
- SQL-based reporting and database management

## Project Files

```
├── analysis.ipynb          # Complete data analysis workflow
├── generate_data.py        # Synthetic data generation
├── sales.csv              # Sample retail sales dataset
└── README.md              # Project documentation
```

## Getting Started

**Prerequisites:** Python 3.7+ with pip package manager

**Installation:**
```bash
pip install pandas matplotlib numpy jupyter
```

**Usage:**
1. Generate the sample dataset:
   ```bash
   python generate_data.py
   ```

2. Launch the analysis notebook:
   ```bash
   jupyter notebook analysis.ipynb
   ```

## Key Insights

The analysis reveals important business metrics from 1,500 sales transactions:

**Financial Performance:**
- Total Revenue: $576,205
- Average Order Value: $384
- Revenue Growth: Month-over-month tracking across 13 months

**Product Performance:**
- Electronics leads with 62% of total revenue
- Home goods contribute 15% of revenue
- Five distinct product categories analyzed

**Data Quality:**
- Complete dataset with no missing values
- 475 unique customers across 25 products
- 365-day transaction history
